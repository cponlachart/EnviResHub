# EnviResHub — ระบบจองทรัพยากรงานวิจัย

ระบบจองห้องประชุม/ห้องเรียน ครุภัณฑ์ เครื่องมือวิทยาศาสตร์ และยานพาหนะ ในที่เดียว
สำหรับสถาบัน/หน่วยงานวิจัยระดับคณะ — รองรับภาษาไทย/อังกฤษ และใช้งานบนมือถือได้

> โปรเจกต์นี้เป็น **prototype (front-end)** ทำงานในเบราว์เซอร์ล้วน ยังไม่มี backend จริง
> ข้อมูลทั้งหมดเป็นตัวอย่างและรีเซ็ตเมื่อรีเฟรช

---

## 🚀 Deploy บน GitHub Pages

1. สร้าง repository ใหม่บน GitHub (Public)
2. อัปโหลดไฟล์ **`index.html`** เข้า repo (ไฟล์นี้รวมทุกอย่างไว้ในไฟล์เดียว เปิดออฟไลน์ได้)
3. ไปที่ **Settings → Pages** → เลือก Branch `main` + โฟลเดอร์ `/ (root)` → **Save**
4. รอ ~1 นาที เว็บจะออนไลน์ที่
   `https://<username>.github.io/<repo-name>/`

> อัปเดตทีหลัง: แก้ที่ไฟล์ต้นฉบับ → สร้าง `index.html` ใหม่ → นำไป replace ใน repo

## 🧭 ฟีเจอร์

**ฝั่งผู้ใช้**
- จองทรัพยากร — แท็บแยกหมวด, ค้นหา + ตัวกรอง (ความว่าง/อาคาร/ความจุ), สลับมุมมอง 3 แบบ
- หน้ารายละเอียด + **ปฏิทินรวมรายสัปดาห์** เลือกช่วงเวลา, จองประจำ (ครั้งเดียว/รายสัปดาห์/รายวัน)
- **ทริป** — รวมหลายทรัพยากรจองพร้อมกัน
- การจองของฉัน — กำลังจะถึง/รออนุมัติ/ประวัติ + QR เช็คอิน + เลื่อนเวลา/ยกเลิก

**ฝั่งผู้ดูแล**
- ภาพรวมผู้บริหาร (utilization, demand vs supply, ต้นทุน, ข้อเสนอแนะ)
- แดชบอร์ด (KPI, กราฟ, heatmap, คิวอนุมัติ + bulk approve)
- จัดการทรัพยากร (เพิ่ม/แก้/ลบ/ซ่อน, สลับสถานะ ว่าง/ไม่ว่าง/ปิดปรับปรุง)
- จัดการผู้ใช้ (บทบาท, โควต้า, ระงับสิทธิ์)
- รายงาน + ตั้งค่าระบบ + Audit Log

**สถานะทรัพยากร**
- 🟢 ว่าง · 🟠 ไม่ว่าง · 🔴 ปิดปรับปรุง

## 🗺️ Roadmap

- **Phase 1 (เสร็จแล้ว)** — ปฏิทินรวม, จองประจำ, ทริป, เลื่อนเวลา, bulk approve
- **Phase 2** — แนะนำทรัพยากรทดแทน, no-show auto-release, พยากรณ์ peak, โควต้า/กฎเวลาทำการ
- **Phase 3 (backend จริง)** — เชื่อม Google Sheets (Apps Script) → Firebase/Supabase, login SSO, แจ้งเตือน Email/LINE

## 🔌 แนวทางต่อ backend ด้วย Google Sheets

โครงสร้างชีตที่แนะนำ (1 ไฟล์ หลายชีต):

| Sheet | คอลัมน์หลัก |
|---|---|
| `Resources` | id, ประเภท, ชื่อ, อาคาร, ความจุ, สถานะ, ต้องอนุมัติ |
| `Bookings` | id, resourceId, userId, วันที่, ช่วงเวลา, สถานะ, รหัสเช็คอิน |
| `Users` | id, ชื่อ, หน่วยงาน, บทบาท, โควต้า, สถานะ |
| `AuditLog` | เวลา, ผู้ใช้, การกระทำ, ประเภท |
| `Settings` | key, value |

ต่อผ่าน **Google Apps Script Web App** (อ่าน/เขียน Sheet แล้วให้หน้าเว็บเรียกผ่าน `fetch`)
ใช้ `LockService` กันการจองช่องเดียวกันพร้อมกัน
